# PRD: 역기획서 생성기 (Reverse PRD Generator)

## 1. 개요

### 배경 및 목적
PM/기획자가 경쟁사·벤치마킹 대상 웹서비스를 분석할 때 수동으로 역기획서를 작성하는 데 수 시간이 소요된다. URL 하나를 입력하면 AI가 자동으로 UI/UX 분석, 기능 명세, 비즈니스 로직, 와이어프레임을 포함한 역기획서를 생성해 분석 시간을 대폭 단축한다.

### 목표
- URL 입력 후 **1분 이내**에 역기획서 초안 생성
- PM이 바로 활용 가능한 수준의 구조화된 문서 제공
- 마크다운 다운로드 및 복사 기능으로 실무 활용성 확보

### 범위 (MVP)
- 공개 접근 가능한 웹사이트 랜딩 페이지 분석
- 로그인이 필요한 페이지, 크롤링 차단(403) 사이트 제외

---

## 2. 사용자 정의

### 주요 페르소나
| 페르소나 | 설명 | 핵심 니즈 |
|---------|------|---------|
| PM/기획자 | 경쟁사 서비스를 빠르게 분석해야 하는 실무자 | 빠른 초안 생성, 구조화된 문서 |
| UX 디자이너 | 벤치마킹 사이트의 UI 패턴을 파악하고 싶은 디자이너 | 와이어프레임, UI 컴포넌트 분석 |
| 스타트업 창업자 | 유사 서비스를 빠르게 분석해 사업 방향을 잡으려는 초기 창업자 | 비즈니스 모델, 차별점 분석 |

## 3. 핵심 기능

### 3-1. URL 크롤링
- **설명**: 입력된 URL의 HTML을 수집하여 텍스트, 메타데이터, 네비게이션 구조 추출
- **기술**: httpx + BeautifulSoup
- **예외 처리**: 403/404/타임아웃 등 에러 시 사용자 친화적 메시지 표시

### 3-2. AI 역기획서 생성
- **설명**: 수집된 데이터를 AI가 분석하여 6개 섹션의 역기획서 마크다운 생성
- **기술**: Google Gemini API (gemini-flash-latest, REST 직접 호출)
- **출력 섹션**:
  1. 서비스 개요 (가치 제안, 타겟 사용자, 비즈니스 모델)
  2. UI/UX 분석 (와이어프레임, 레이아웃, 컴포넌트, 사용자 플로우)
  3. 기능 명세 (7개 이상, 우선순위 포함)
  4. 비즈니스 로직 추정
  5. 경쟁우위 / 차별점
  6. 기술 스택 추정

### 3-3. 실제 스크린샷 제공
- **설명**: Google PageSpeed Insights API로 대상 사이트 스크린샷 자동 캡처
- **표시**: 역기획서 상단에 실제 화면 이미지 표시

### 3-4. AI 와이어프레임 생성
- **설명**: Gemini가 HTML+CSS 와이어프레임 코드 생성, iframe으로 렌더링
- **특징**: 링크 클릭 비활성화, 컨테이너 너비에 맞게 자동 축소

### 3-5. 결과 활용 기능
- 마크다운 복사 버튼
- 파일 다운로드 (`.md` 형식)
- 섹션 접기/펼치기

## 4. 비기능 요구사항

| 항목 | 목표 |
|------|------|
| 응답 시간 | 역기획서 생성 60초 이내 |
| 에러 처리 | 모든 오류를 사용자 친화적 메시지로 표시 |
| 호환성 | 최신 Chrome/Edge 브라우저 |
| 배포 | Render 무료 티어 (Docker) |

## 5. 기술 스택

| 영역 | 기술 |
|------|------|
| 백엔드 | Python FastAPI |
| 프론트엔드 | HTML + CSS + Vanilla JS |
| AI | Google Gemini API (gemini-flash-latest) |
| 크롤링 | httpx + BeautifulSoup |
| 스크린샷 | Google PageSpeed Insights API |
| 마크다운 렌더링 | marked.js (CDN) |
| 배포 | Render (Docker) |

## 7. 성공 지표

- 역기획서 생성 성공률 80% 이상 (공개 사이트 기준)
- 생성 소요 시간 평균 60초 이내
- 해커톤 데모에서 3개 이상 사이트 실시간 생성 성공
